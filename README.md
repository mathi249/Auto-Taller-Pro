# ⚙ AutoTaller Pro

Sistema de gestión para talleres mecánicos. Registrá órdenes de trabajo, arreglos, horas y generá reportes.

---

## 🚀 Cómo subir a Vercel (gratis, 10 minutos)

### Opción A — Sin código (más fácil)

1. Creá una cuenta gratis en [vercel.com](https://vercel.com)
2. Comprimí esta carpeta en un ZIP
3. En Vercel → **"Add New Project"** → **"Upload"** → subí el ZIP
4. Hacé clic en **Deploy** → ¡listo!

### Opción B — Con GitHub (recomendada)

1. Creá una cuenta en [github.com](https://github.com) si no tenés
2. Creá un repositorio nuevo
3. Subí los archivos de esta carpeta
4. En [vercel.com](https://vercel.com) → **"Add New Project"** → conectá tu GitHub → seleccioná el repo
5. Clic en **Deploy**

Tu app queda en: `https://tu-proyecto.vercel.app`

---

## 📱 Instalar en el celular (sin App Store)

1. Abrí el link de tu app en Chrome (Android) o Safari (iPhone)
2. Tocá el menú (⋮ o compartir)
3. Seleccioná **"Añadir a pantalla de inicio"**
4. ¡Funciona como una app nativa!

---

## ✨ Funciones incluidas

- ✅ Registrar órdenes por cliente, auto, patente, fecha
- 🔧 Anotar cada arreglo con descripción, materiales y horas
- 📊 Panel con estadísticas (órdenes, horas, ingresos)
- 🔍 Búsqueda y filtro por estado
- ☰ Vista lista y ⊞ vista tarjetas
- 🖨 Imprimir/descargar reporte por orden
- 📄 Reporte general de todas las órdenes
- 💾 Datos guardados automáticamente en el navegador
- 📱 Compatible con celular y tablet
